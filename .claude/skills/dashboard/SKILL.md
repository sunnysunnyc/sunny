# 경쟁 분석 대시보드

경쟁사 데이터를 읽어 인터랙티브 HTML 대시보드를 생성한다.

## 프로세스

### Step 1: 데이터 확인
`data/competitors/` 디렉토리의 모든 JSON 파일을 확인하고, 데이터가 7일 이상 오래된 경쟁사가 있으면 경고한다.

### Step 2: 대시보드 생성
아래 명령어로 대시보드 스크립트를 실행한다:

```bash
python3 scripts/generate-dashboard.py
```

### Step 3: 대시보드 열기
생성된 대시보드를 브라우저에서 연다:

```bash
open reports/dashboard.html
```

### Step 4: 결과 안내
사용자에게 대시보드가 생성되었음을 알리고, 대시보드에 포함된 섹션을 안내한다:

1. **경쟁사 개요 카드**: 각 경쟁사 MAU, 수익원, SWOT 요약
2. **기능 비교 매트릭스**: 카테고리별 기능 보유 현황
3. **포지셔닝 맵**: 기능 범위 x 과금 강도 2x2 매트릭스
4. **SWOT 분석**: 탭 전환으로 경쟁사별 SWOT 상세
5. **최근 동향 타임라인**: 시간순 주요 이벤트

대시보드는 자체 완결형 HTML이므로 별도 서버 없이 브라우저에서 바로 열 수 있다.
