# 시장 동향 업데이트 스킬

## 분석 프로세스

### Step 1: 최신 뉴스 수집
아래 검색어로 웹 검색을 실행한다 (한국어 + 영어 모두):

1. "팬덤 플랫폼 시장 동향 2026"
2. "K-pop fan platform market 2026"
3. "위버스 버블 베리즈 경쟁 최신"
4. "fandom app industry news"
5. "팬덤 커뮤니티 앱 투자 인수합병"
6. "카카오 베리즈 Berriz 팬덤 플랫폼"
7. "아이돌 팬소통 앱 트렌드"

### Step 2: 뉴스 분류 및 분석
수집된 정보를 아래 카테고리로 분류한다:

| 카테고리 | 설명 |
|---------|------|
| 📦 제품/기능 | 새 기능 출시, 앱 업데이트 |
| 💰 사업/재무 | 실적, 투자, M&A |
| 👥 사용자 | MAU 변화, 팬 반응, 리뷰 트렌드 |
| 🏢 조직/인사 | 인력 변동, 조직 개편 |
| 📊 시장/산업 | 전체 시장 규모, 규제, 기술 트렌드 |
| 🤝 파트너십 | 아티스트 영입, 제휴, 플랫폼 간 이동 |

### Step 3: 경쟁사 데이터 업데이트
뉴스에서 특정 경쟁사 관련 정보가 발견되면:
1. 해당 `data/competitors/[이름].json` 파일을 읽는다
2. `trends` 배열에 새 항목을 추가한다 (기존 항목은 유지, append-only)
3. 해당되는 다른 필드도 업데이트한다 (MAU, 재무 등 변경 시)
4. `meta.last_updated`를 오늘 날짜로 갱신한다
5. `meta.sources`에 새 출처를 추가한다
6. 파일을 저장한다

### Step 4: 시장 브리핑 출력
아래 형식으로 브리핑을 출력한다:

```
## 📰 팬덤 플랫폼 시장 브리핑 - [오늘 날짜]

### 🔥 주요 헤드라인
1. [가장 임팩트 있는 뉴스]
2. [두 번째 주요 뉴스]
3. [세 번째 주요 뉴스]

### 카테고리별 동향

#### 📦 제품/기능
- ...

#### 💰 사업/재무
- ...

#### 👥 사용자 동향
- ...

#### 🤝 파트너십
- ...

### 💡 전략적 시사점
- [시사점 1]: 근거와 함께
- [시사점 2]: 근거와 함께

### ⏭️ 주목할 포인트
- 앞으로 주시해야 할 이벤트/발표

### 📋 데이터 업데이트 내역
- [경쟁사명].json: [업데이트된 필드] 갱신됨
```

## 주의사항
- 확인되지 않은 정보는 반드시 ⚠️ 표시와 함께 "(미확인)" 태그를 붙인다
- 뉴스의 날짜를 명확히 표기하여 최신성을 판단할 수 있게 한다
- 단순 나열이 아닌, "우리에게 어떤 의미인가?"를 반드시 포함한다
- 검색 결과가 부족한 카테고리는 "이 기간 중 주요 변화 없음"으로 명시한다
- 웹 검색 결과의 출처를 가능한 한 링크와 함께 제공한다
