#### 2-2. 입시·데뷔 경쟁률 자체도 살벌

- **입시 (2026)**: 홍익대 실용음악(싱어송라이터) **171.5:1** / 서경대 **163.7:1** / 한예종 연극원 **148.6:1**
- 부모의 결정적 페인: "데뷔도 못하고 학력 공백이 비면 어떡해?"

#### 2-3. 시장 학원들의 4대 문제 (경쟁사 비판)

1. **허위/과장 합격률 마케팅** (96%·97%·99%·명단 나열식)
2. **'내방 오디션' 과도한 특권화** — 사실 규모 있는 학원은 다 함
3. **공장형 단기 카피 커리큘럼** (고음·유명 안무 카피)

→ **결정타**: 데뷔도 입시도 살벌하고, 시장 학원들은 단기 합격을 미끼로 삼을 뿐 **평생 자생력을 만들어주는 곳이 없다.**

### #3 Solution — 4대 솔루션 영역 + 22+ 차별화 수업

**메시지**: "Skill Factory가 아니라 Artist Incubator. 4대 영역으로 평생 자생력을 만든다."

#### 4대 솔루션 영역

| 영역 | 핵심 |
|---|---|
| ① **다각적 진로 설계 & 직접 캐스팅** | 오디션 & 입시 듀얼 패스 / 올라운더 융합 / 원석 직접 발굴 |
| ② **실전 역량 & 자생력 강화** (Artist Self-Reliance) | 창작 역량 / 스피치·지성 / 비주얼 브랜딩 (셀프 메이크업·체형·성형 협업) |
| ③ **인성·사회성 시스템화** (Social & Emotional Intelligence) | 순환 리더제 / 상담사 상주·감정 일기 / **데이터 기반 아카이빙** = "압도적 포트폴리오" |
| ④ **글로벌 몰입 & 운영 시스템** | 이중 언어 몰입 / 글로벌 인바운드 K-POP 유학 패키지 / 전담 매니저 |

#### 22+ 차별화 수업 (5축)

- **인문·언어**: 국어(문해력) / 스피킹(논술·MC·인터뷰·비즈니스 매너) / 랩(딕션) / 영어(K-드라마 활용)
- **무대·카메라**: 무대장악력 / 모델 포징·워킹(2주 1회 자체 촬영실) / 음향·사운드 엔지니어
- **제작·뒷무대**: 공연 연출·감독 역할 / 영상 제작·편집 / 화성학·시창청음
- **IP·마케팅**: 퍼스널 브랜딩·스타일링 / 메이크업 / 소셜미디어 알고리즘 / 라이브 커머스 / 협상 커뮤니케이션
- **창작**: 작사·극본 작가 / 연기 / 뮤지컬 / 스케치(디자인) / 퍼포먼스 댄스 디렉팅 / **보컬 하모니·댄스 하모니** (어반송라이팅·어반댄스 라이팅) / 통기타·드럼 퍼포먼스

→ **5개 학과 입시 매핑**도 그대로 (실용음악·실용무용·연극영화·방송연예·방송영상). 하지만 본질은 *입시·데뷔의 표면 너머에 있는 22+ 융합 수업*.

### #4 Why Now

**메시지**: "산업·인재상·교육 시장·자본 시장 4중 변화가 지금 일어나고 있다."

1. **K-콘텐츠 산업 폭발 성장**:
   - 한국 콘텐츠 산업 매출 **161조** (2024, +2.6%)
   - **한국 대중문화예술산업 매출 15조 3,845억** (2024, **+34.5% vs 2022**)
   - 음악 산업 +15.8% / 음악 수출 +32.4%
   - K-pop 팬덤 비즈니스 3조 → 10조 (2024 → 2029 전망)

2. **기획사 R&D의 외주화** ⭐ (가장 강력한 무기):
   - 대중문화예술기획업체 **4,471개로 증가**
   - 기획사 소속 연습생 **-17.7% 감소**
   - 기획사가 직접 키우지 않고 **학원에서 검증된 인재 선발** 추세
   - 원밀리언이 SM·YG·JYP에 안무 공급
   - → **외부 전문 학원의 시장이 열림**

3. **4세대 K-pop 인재상 변화**: 자작곡·셀프 프로듀싱·자체 콘텐츠·퍼스널 브랜딩까지 요구 → 종합 트레이닝 필수

4. **사교육·교육 시장 양극화**:
   - 국내 사설 학원 시장 **29조원**
   - 학령인구 감소에도 사교육 우상향
   - 사교육 참여학생 월평균 **60만원 돌파** (2025)
   - **보컬·댄스 학원 시장 연평균 20% 성장** (~2035)
   - 글로벌 온라인 댄스 교육 **2035년 약 180억 달러**
   - AI 동작 분석 47% 도입 / VR/AR 몰입 / 데이터 기반 코칭

5. **자본 시장의 학원 비즈니스 매력도**: 메가스터디·시대인재 등 학원 체인의 자본시장 매력도 검증 → 학원 사업의 투자 유치 가능성

### #9 Business Model — **3-Tier 가격 구조** ⭐

**메시지**: "단순반·베네핏·프리미엄 3단계로 객단가를 구조화한다."

| 등급 | 월 객단가 | 매니저 | 운영 |
